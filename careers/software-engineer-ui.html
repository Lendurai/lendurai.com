Software Engineer, User Interface

Tallinn, Estonia (Hybrid)

About the company and the role

At Lendurai, we enable unmanned aerial vehicles (UAVs) to operate independently of global navigation satellite systems and radio control, using computer vision and machine learning. We partner with multiple UAV manufacturers to deliver cost-effective capabilities to public sector and defence customers. Our team has deep experience in aerospace engineering, defence tech, and autonomous systems, with backgrounds at companies like Bolt (EE), Open Cosmos (UK), and Starship Technologies (EE).

As a UI Software Engineer, you will be responsible for developing intuitive and responsive user interfaces for our drone control and mission planning systems. You'll work closely with UX designers and backend engineers to create seamless experiences that enable operators to effectively control and monitor autonomous drone operations.

What you will be doing

  • Design and implement responsive web interfaces for drone control and mission planning systems
  • Create real-time data visualization components for flight telemetry and mission status
  • Develop intuitive map-based interfaces for mission planning and execution
  • Implement robust error handling and user feedback mechanisms
  • Optimize UI performance for real-time updates and low-latency interactions
  • Work with UX designers to implement and iterate on user interface designs
  • Collaborate with backend engineers to integrate APIs and real-time data streams

We are looking for

Required skills:

  • 3+ years of experience in frontend development
  • Strong proficiency in TypeScript and modern JavaScript
  • Experience with React or similar modern frontend frameworks
  • Knowledge of state management solutions (Redux, MobX, etc.)
  • Experience with real-time data visualization libraries
  • Understanding of WebSocket communication
  • Strong focus on performance optimization and responsive design

Nice to have:

  • Experience with mapping libraries (Mapbox, Leaflet, etc.)
  • Background in developing interfaces for control systems
  • Knowledge of WebGL or Three.js for 3D visualization
  • Experience with real-time data streaming and visualization
  • Understanding of UX principles and accessibility standards

  • Competitive salary and equity package
  • Opportunity to shape the future of autonomous drone interfaces
  • Modern tech stack and focus on best practices
  • Professional development opportunities
  • Flexible working hours
  • Health insurance
  • Regular team events
